# IoT-Smart-home-system-prototype
An IoT-based smart home automation system using ESP8266 and Arduino Uno. Features web-controlled relay, LED, servo motor, and smart fan with 3 modes (manual, temperature, gas sensor). Includes DHT22, PIR motion, flame, LDR, MQ-2 gas sensor, and RFID reader. Real-time monitoring via WiFi web interface.
